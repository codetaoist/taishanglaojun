# 路线图与交付物清单

按五阶段推进，交付物明确到文件与可运行成果。[0]

## 阶段一：准备期
- 交付：Monorepo 结构、核心文档、OpenAPI 草案、CI/CD 雏形。

## 阶段二：核心功能期
- 交付：后端基础 API（健康检查、插件管理端点），管理后台页面骨架。

## 阶段三：功能扩展期
- 交付：前端核心页面完善、后端业务模块扩展、数据模型深化与索引优化。

## 阶段四：插件生态期
- 交付：插件市场管理与全生命周期 CI/CD 完整打通，审计与日志完善。

## 阶段五：稳定交付期
- 交付：部署脚本与环境基线完善、监控告警、性能与安全加固、回滚与演练。

## 文件清单（示例）
- `docs/*` 全部文档与流程图（含 mermaid）。
- `openapi/openapi.yaml` 接口事实源。
- `db/schema.sql` 数据模型与初始化脚本。
- （后续）`apps/admin` 与 `services/api` 可运行最小版本。
