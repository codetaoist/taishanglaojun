# 太上老君移动端应用

## 项目概述

本目录包含太上老君系统的移动端应用，支持多平台原生开发：

- **iOS应用** (Swift) - 支持iPhone和iPad
- **Android应用** (Kotlin/Java) - 支持Android设备
- **鸿蒙应用** (ArkTS) - 支持华为鸿蒙系统

## 核心功能

### 🗺️ 位置追踪系统
- **实时GPS定位**：持续获取用户位置信息
- **轨迹记录**：完整记录用户每日移动路径
- **历史查询**：按日期查看历史轨迹数据
- **后台运行**：确保位置服务持续工作
- **隐私保护**：符合各平台隐私规范

### 📱 平台特性
- **iOS**：使用Core Location框架，支持后台位置更新
- **Android**：使用Fused Location Provider，前台服务保持运行
- **鸿蒙**：使用位置服务API，后台任务管理

### 🔒 安全特性
- **数据加密**：本地存储和网络传输加密
- **权限管理**：严格的位置权限申请流程
- **隐私合规**：符合GDPR、个人信息保护法等规范

## 项目结构

```
mobile-apps/

├── android/               # Android应用 (Kotlin)

└── docs/                  # 开发文档
```

## 开发环境要求

### iOS开发
- Xcode 15.0+
- iOS 14.0+
- Swift 5.9+

### Android开发
- Android Studio 2023.1+
- Android API 24+ (Android 7.0)
- Kotlin 1.9+

### 鸿蒙开发
- DevEco Studio 4.0+
- HarmonyOS API 9+
- ArkTS

## 快速开始

1. 克隆项目到本地
2. 选择对应平台目录
3. 按照各平台README进行环境配置
4. 运行项目

## API集成

移动端应用将与后端API进行集成：
- 位置数据上传
- 用户认证
- 轨迹数据同步
- 设置配置同步

## 隐私声明

本应用严格遵守各平台隐私政策：
- 仅在用户明确同意后收集位置信息
- 位置数据仅用于轨迹记录功能
- 支持用户随时删除位置数据
- 不会将位置信息分享给第三方
