# 太上老君AI平台快速开始指南

## 欢迎使用太上老君AI平台

太上老君AI平台是一个集成了先进人工智能技术的综合性平台，提供智能对话、图像生成、文档分析等多种AI服务。本指南将帮助您快速上手使用平台的各项功能。

## 🚀 5分钟快速体验

### 1. 注册账户

访问 [太上老君AI平台](https://taishanglaojun.ai) 并完成注册：

1. 点击"注册"按钮
2. 填写邮箱和密码
3. 验证邮箱地址
4. 完成个人信息设置

### 2. 开始第一次对话

注册完成后，您可以立即开始使用智能对话功能：

1. 在主界面找到对话输入框
2. 输入您的问题，例如："你好，请介绍一下你的功能"
3. 点击发送按钮
4. 等待AI回复

### 3. 尝试图像生成

体验强大的AI图像生成功能：

1. 点击左侧菜单的"图像生成"
2. 在提示词输入框中描述您想要的图像，例如："一只可爱的小猫坐在花园里"
3. 点击"生成图像"按钮
4. 等待图像生成完成

### 4. 上传文档分析

测试文档分析功能：

1. 点击"文档分析"菜单
2. 上传一个PDF或Word文档
3. 选择分析类型（摘要、关键词提取等）
4. 查看分析结果

## 🔧 开发者快速开始

### 获取API密钥

1. 登录平台后进入"设置" → "API设置"
2. 点击"创建新密钥"
3. 设置密钥名称和权限
4. 复制生成的API密钥

### 第一个API调用

```bash
# 测试API连接
curl -X POST https://api.taishanglaojun.ai/v1/chat/messages \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "content": "Hello, World!",
    "model": "gpt-4"
  }'
```

### SDK安装

#### JavaScript/Node.js
```bash
npm install @taishanglaojun/api-client
```

```javascript
import { TaishanglaojunClient } from '@taishanglaojun/api-client';

const client = new TaishanglaojunClient({
  apiKey: 'your-api-key'
});

const response = await client.chat.sendMessage({
  content: '你好，世界！'
});
```

#### Python
```bash
pip install taishanglaojun-api
```

```python
from taishanglaojun import TaishanglaojunClient

client = TaishanglaojunClient(api_key="your-api-key")
response = client.chat.send_message(content="你好，世界！")
```
